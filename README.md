# Aarambh - Emotional Support App for Teens

## Overview
Aarambh is an interactive Streamlit app designed to provide emotional support to teenagers. Users can share their feelings and receive helpful, categorized responses to guide them.

## Features
- Supports 20+ emotions with 7000+ responses
- Auto-detect emotion using NLP (TextBlob)
- Tracks user mood over time with visualizations
- Provides actionable tips and encouragement

## Technologies Used
- Python, Streamlit
- Pandas, Matplotlib, TextBlob

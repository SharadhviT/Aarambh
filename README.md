# 💛 Aarambh AI+

An AI-powered emotional intelligence system designed to support teenage mental well-being.

- Real-time AI chatbot (LLM-based)
- Context-aware emotional conversations
- Mood detection and tracking
- Behavioral analytics dashboard
- Emotional stability insights
- Personalized self-care recommendations

- Pandas
- Plotly
